All the details of datalake solution project: Designing an Enterprise DataLake System.

There is an EMR compmnay which have 3 layererd application to provide analytical solution to the medical facilities including hospitals, day clinics, and urgent care clinics.

Comapny has all their data stored in traditonal sql database, but has hit the point where it has became extremely complex to manage this data, and has faced service outage becuase of the spike in data which broke ETL processes, and analytics were impacted thus causing customer issues with their business.

Goal of this project is to create a datalake based solution for managing allt he companies data, preferably using open source tools.
